Proyectos Destacados

Sistema de Geston de Registro Medico Electronico (RME)

Diseñe e implemente un sistema de gestion de Registro Medico Electronico utilizando Java y PostgreSQL. El sistema permite almacenar, buscar y gestionar informacion medica de pacientes de manera segura y eficiente. El proyecto incluye la implementacion de estructuras avanzadas como Arboles B+ y Arboles AVL para optimizar el acceso a los datos.
